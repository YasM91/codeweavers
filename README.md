## Codeweavers Frontend Task

This is a Next application that uses React, Tailwind and Typescript. It has tests using React Testing Library.

Requires node 20.19+ for the application to run.

Run `npm ci` to install dependencies.

Run `npm run dev` to start the application.

Run `npm run test` to test the application.

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
